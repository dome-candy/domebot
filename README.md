## Installation

using HTTPS
```sh
$ git clone https://github.com/autoboxer/domebot.git
```

using SSH
```sh
$ git clone git@github.com:autoboxer/domebot.git
```

## Usage

if you're connecting to mongo DB locally, be sure to start it using
```sh
$ mongod
```

launch the program
```sh
$ npm start
```
